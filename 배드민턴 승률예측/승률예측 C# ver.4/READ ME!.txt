이미 설치되어 있다면 OK!

★★★간략한 프로그램 실행 순서★★★
1. 배트민턴 승률예측 프로그램.exe 파일로 실행
2. 기본 정보 클릭   -> 단식, 복식에 따라 정보 입력 후 "입력" 버튼 클릭
		-> "입력되었습니다." 메시지 박스 확인
3. 승률 예측 클릭
4. 결과 보조창 클릭
5. '배드민턴 승률 예측 프로그램'창에서 버튼을 통해 선수,상황,득점여부를 선택하면
   정보들이 변경됩니다.
6. 하나의 세트가 끝나게 되면 "세트 종료" 버튼 클릭
7. 이후 5~6번 과정 반복.



★★★주의 사항★★★
버그가 많을 수 있습니다. 기능이 정상적으로 작동하지 않는 경우 즉시 연락주세요!!
1. 경기가 끝나고 새로운 선수들의 정보를 입력하고 싶다면 프로그램을 다시 시작해주세요. (버그의 가능성이 있음)
2. 선수들의 사진 넣는방법은 '사진넣는방법.png'를 참고하세요.
3. 세트가 자동으로 종료되지 않습니다! 한 세트가 끝나면 꼭 세트 종료를 눌러주세요!
4. 경기 도중 실수로 결과 보조창을 닫았다면, 다시 "결과 보조창" 버튼을 눌러 창을 띄워주세요.
   다음 득점 여부를 입력하면 정상적으로 돌아올겁니다.
5. 선수 정보를 입력하지 않고 결과 보조창을 실행 후 종료하면 에러가 발생합니다. 프로그램을 다시 시작해주세요.
