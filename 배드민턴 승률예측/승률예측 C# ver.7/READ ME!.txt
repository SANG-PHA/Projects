=> 이전의 값(승률, 사후확률, 분자, 분모 등등)들이 현재의 사후확률에 영향을 미치기 때문에 선택한 항목을 제거하는 기능대신
     마지막 입력을 취소할 수 있도록 한겁니다 ㅜ_ㅜ


★★★간략한 프로그램 실행 순서★★★
1. 배트민턴 승률예측 프로그램.exe 파일로 실행
2. 기본 정보 클릭   -> 단식, 복식에 따라 정보 입력 후 "입력" 버튼 클릭
		-> "입력되었습니다." 메시지 박스 확인
3. 승률 예측 클릭
4. 결과 보조창 클릭
5. '배드민턴 승률 예측 프로그램'창에서 버튼을 통해 선수,상황,득점여부를 선택하면
   정보들이 변경됩니다.
6. 하나의 세트가 끝나게 되면 "세트 종료" 버튼 클릭
7. 이후 5~6번 과정 반복.
